cpp教学大纲-语言基础

  1. C++程序结构与编译环境
    1.1 C++源文件后缀（.cpp, .cc, .cxx）
    1.2 编译过程（预处理、编译、汇编、链接）
    1.3 g++常用编译选项（-o 指定输出， -std=c++11/14/17， -Wall 开启警告）
    1.4 main函数的严格定义（int main() 与 int main(void) 的区别）
    1.5 main函数的返回值（ 0 表示正常退出，非 0 表示异常）
    1.6 大括号 {} 定义代码块与作用域边界
    1.7 每条语句以分号 ; 结尾（空语句的概念）
    1.8 注释的三种形式（//， /* */，以及嵌套规则的注意事项）
    1.9 #include预处理指令（头文件拷贝展开）
    1.10 bits/stdc++.h 万能头（优缺点：编译慢但方便，非标准但竞赛常用）
    1.11 using namespace std; 的原理（命名空间污染风险）

  2. 标准输入输出流
    2.1 cout对象（ostream类实例）
    2.2 cout输出字符串字面量（"Hello"）
    2.3 cout输出变量与表达式
    2.4 cout的连续输出（<< 运算符的链式调用）
    2.5 endl与'\n'的本质区别（endl刷新缓冲区，'\n'仅换行）
    2.6 cin对象（istream类实例）
    2.7 cin读取时的跳过空白字符机制（空格、Tab、换行自动跳过）
    2.8 cin读取字符串遇空格停止的陷阱
    2.9 cin.get() 与 cin.getline() 读取含空格的字符数组
    2.10 getline(cin, stringVar) 读取整行string
    2.11 scanf与printf的占位符（%d, %lld, %f, %lf, %c, %s）
    2.12 scanf务必加 & 取地址（除数组名外）
    2.13 printf的宽度控制（%5d 右对齐占5位， %-5d 左对齐）
    2.14 printf的精度控制（%.2f 保留两位小数，实际是四舍五入）
    2.15 ios::sync_with_stdio(false) 解除C与C++流同步
    2.16 cin.tie(nullptr) 解除cin与cout的绑定（提升读写效率）

  3. 变量、常量与数据类型
    3.1 变量名命名规则（字母、数字、下划线，不能数字开头，不能是关键字）
    3.2 变量的四种存储类型（auto, register, static, extern）
    3.3 变量的定义（分配内存）与声明（告知类型）
    3.4 变量的初始化（拷贝初始化 int a = 5; 与直接初始化 int a(5);）
    3.5 列表初始化 int a{5};（防止窄化转换）
    3.6 常量 const（编译时确定，只读，必须初始化）
    3.7 常量表达式 constexpr（C++11起，强制编译期计算）
    3.8 字面量后缀（10L, 3.14f, "string"s）
    3.9 整数类型 int（通常32位，范围 −2147483648∼2147483647 ）
    3.10 无符号整数 unsigned int（范围 0∼4294967295 ，注意死循环陷阱）
    3.11 长整型 long long（64位，范围约 ±9.22×1018 ）
    3.12 无符号长整型 unsigned long long（范围 0∼1.84×1019 ）
    3.13 字符类型 char（本质是整数，占1字节，范围 −128∼127 或 0∼255 ）
    3.14 有符号与无符号 char（signed char 与 unsigned char）
    3.15 单精度浮点 float（4字节，6-7位有效数字，精度不足问题）
    3.16 双精度浮点 double（8字节，15-16位有效数字）
    3.17 布尔类型 bool（true 为 1 ，false 为 0 ，非零值转为 true）
    3.18 空类型 void（无值，用于函数无返回或泛型指针）
    3.19 sizeof 运算符（编译期计算，返回 size_t 类型）
    3.20 类型别名 typedef 与 using（C++11）

  4. 运算符与表达式
    4.1 算术运算符（+ - * / %）
    4.2 整数除法的向零截断（ 5/2=2 , −5/2=−2 ）
    4.3 取模运算符 % 仅适用于整数，结果的符号与被除数一致
    4.4 关系运算符（> < >= <= == !=）返回布尔值
    4.5 浮点数比较的精度陷阱（避免直接 ==，改用 fabs(a-b) < eps）
    4.6 逻辑运算符（&&, ||, !）
    4.7 逻辑短路求值（&&前面为假则不计算后面，||前面为真则不计算后面）
    4.8 位运算符（& 与， | 或， ^ 异或， ~ 非， << 左移， >> 右移）
    4.9 左移 << 低位补 0 ，相当于乘以 2 的幂（注意符号位）
    4.10 右移 >> 对于无符号数高位补 0 ，对于有符号数高位补符号位（算术右移）
    4.11 赋值运算符 =（右结合性）
    4.12 复合赋值运算符（+=, -=, *=, /=, %=, &=, |=, ^=, <<=, >>=）
    4.13 自增自减（前置 ++i：先自增后返回；后置 i++：先返回后自增）
    4.14 前置与后置的性能差异（后置有临时拷贝，但编译器通常会优化）
    4.15 逗号运算符（顺序执行，返回最后一个表达式的值）
    4.16 条件运算符（三目运算符 ?:，右结合）
    4.17 运算符优先级表记忆技巧（算术 > 关系 > 逻辑，位运算介于算术与关系之间）
    4.18 括号 () 强制改变优先级与结合性

  5. 流程控制
    5.1 if 语句（仅条件为真执行）
    5.2 if-else 语句（二选一）
    5.3 else if 链（多选一，注意条件的互斥与顺序）
    5.4 悬空 else 问题（else 与最近的未匹配 if 配对）
    5.5 switch 语句（整型或枚举的跳转）
    5.6 switch 中 case 穿透（必须加 break，否则继续执行）
    5.7 switch 的 default 分支（处理其他所有情况）
    5.8 while 循环（先判断后执行，可能一次都不执行）
    5.9 do-while 循环（先执行后判断，至少执行一次）
    5.10 for 循环（初始化; 条件; 变化）
    5.11 for 循环的初始化部分可定义局部变量（作用域仅限于循环内）
    5.12 范围 for 循环（C++11，for(int x : arr) 遍历容器）
    5.13 范围 for 的引用修改（for(int &x : arr) 可修改原值）
    5.14 break 语句（跳出当前最近的一层循环或 switch）
    5.15 continue 语句（跳过本次循环剩余代码，进入下一次迭代）
    5.16 goto 语句（破坏程序结构，仅可用于多层循环跳出）

  6. 函数
    6.1 函数声明（原型）与定义（实现）分离
    6.2 函数返回类型与 return 语句（返回值的拷贝或移动）
    6.3 无返回值函数（void，可以无 return 或 return;）
    6.4 形参（形式参数）与实参（实际参数）
    6.5 值传递（拷贝传参，修改不影响实参）
    6.6 地址传递（指针传递，通过解引用修改实参）
    6.7 引用传递（C++特性，别名，直接操作实参，避免拷贝）
    6.8 常引用传递（const T &，只读且高效，可绑定临时量）
    6.9 函数默认参数（从右向左依次缺省）
    6.10 函数占位参数（只写类型不写名称，用于兼容）
    6.11 函数重载（同名不同参，参数个数或类型不同）
    6.12 函数重载的匹配规则（精确匹配 > 提升转换 > 标准转换）
    6.13 内联函数 inline（建议编译器展开，减少调用开销，但代码膨胀）
    6.14 函数指针（指向函数的指针，回调机制）

  7. 作用域与存储类别
    7.1 块作用域（{} 内定义，出了花括号立即销毁）
    7.2 局部变量（栈内存，自动存储期，未初始化则含随机值）
    7.3 静态局部变量（static，存储在静态区，只初始化一次，生命周期贯穿程序）
    7.4 全局变量（外部链接，所有文件可见，需加 extern 声明）
    7.5 静态全局变量（internal linkage，仅本文件可见）
    7.6 命名空间（namespace）解决全局命名冲突
    7.7 匿名命名空间（相当于加了 static 的内部链接）
    7.8 :: 作用域解析运算符（访问全局变量或命名空间成员）

  8. 数组、字符串与复合类型
    8.1 一维数组的定义（type name[size]）
    8.2 数组大小必须是编译期常量（C++不支持变长数组VLA）
    8.3 数组的初始化列表（int a[5] = {1,2,3}; 未初始化的补 0 ）
    8.4 数组名是常量指针（指向首元素地址）
    8.5 下标运算符 [] 的本质（指针偏移：a[3] 等价于 *(a+3)）
    8.6 数组越界的危害（未定义行为，覆盖其他变量或导致段错误）
    8.7 二维数组的连续存储（行优先存储）
    8.8 字符数组（char str[20]）
    8.9 字符串字面量末尾隐含 '\0'（空终止符）
    8.10 C风格字符串函数（strlen, strcpy, strcat, strcmp）需包含 <cstring>
    8.11 std::string 类的优势（自动管理内存）
    8.12 string 的容量与大小（size(), length(), capacity()）
    8.13 string 的查找（find, rfind, find_first_of）
    8.14 string 的子串（substr(pos, count)）
    8.15 string 的修改（insert, erase, replace, append）
    8.16 结构体 struct（成员默认公有）
    8.17 结构体的内存对齐（字节对齐规则，sizeof 可能大于成员总和）
    8.18 联合体 union（成员共享内存，同一时间只存一个值）
    8.19 枚举 enum（枚举量是常量，隐式转为 int）
    8.20 强枚举类型 enum class（C++11，作用域限定，不可隐式转换）

  9. 指针与动态内存
    9.1 指针的本质（存储变量的内存地址）
    9.2 取地址运算符 &（获取变量地址）
    9.3 解引用运算符 *（通过地址访问值）
    9.4 空指针 nullptr（C++11，类型安全，代替 NULL）
    9.5 指针与数组的等价关系（但指针可以移动，数组名不能）
    9.6 动态内存分配（new 与 delete）
    9.7 动态数组分配（new int[n]，释放用 delete[]）
    9.8 智能指针（内存管理进阶，暂不深入）

  10. STL入门
    10.1 vector 动态数组的本质（可变长，连续存储）
    10.2 vector 的构造（vector<int> v(n), vector<int> v = {1,2,3}）
    10.3 vector 的尾部操作（push_back, pop_back, emplace_back）
    10.4 vector 的容量管理（reserve 预留空间，resize 改变大小）
    10.5 stack 容器适配器（后进先出 LIFO）
    10.6 stack 核心操作（push, pop, top, empty）
    10.7 queue 容器适配器（先进先出 FIFO）
    10.8 queue 核心操作（push, pop, front, back, empty）
    10.9 pair 键值对（make_pair, .first, .second）
    10.10 迭代器概念（begin(), end(), 正向遍历）
    10.11 范围 for 循环遍历 STL 容器（底层依赖迭代器）
